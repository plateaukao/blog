剛好今天（週六）因為大部分行業要補上班，  
所以捷運還是有通。  
但是非上班時間的搭車人數就相當地少。

應該不多吧（爬山除外）。  
這地方，實在是不適合一個人來啊。  
放眼望去，不是情侶就是家庭。  
  
\*\*\*\*  
  
對於現在比較常用的兩隻手機（Diamond, Touch HD)，  
我一直有個不滿意的地方。  
Diamond明明下面有五向鍵和四個硬體按鍵，  
偏偏能customize的鍵就只有通話鍵長按。  
End key在沒有通話的時候，作用就如同那個Home key，  
真搞不懂多弄出一顆Home key的好處在哪。  
就因為Manila在Today Screen，就專門為它生了一個鍵嗎？  
就算是如此，  
那至少也該有個tool可以讓user自己設定Home key的用途。  
畢竟，比較在意速度的人，應該會把Manila關掉吧。  
這麼一來，那顆Home key可能一點兒用處也沒有。  
  
再來是那顆Back key。有一顆鍵可以跳回上一個軟體，  
故然是件好事，但是為什麼就不能一次到位，  
弄個Alt-Tab的軟體，  
讓user按一下是回前一個軟體，  
連續多按幾下會出現畫面讓使用者可以在已經開啟的程式中做切換。  
難不成當初預期使用者一次就只會開兩個程式而已。  
明明不是feature phone的行為模式，可是偏要生一個像是feature phone才有的back key...  
  
講了那麼多，  
結論就是，我平常只會用到通話鍵和結束鍵。  
上面提到的兩個鍵對我來說都是雞肋，  
它們的存在對我來說純粹是佔位置而已。  
  
之前同事有問到在PocketPC上有沒有可以快速切換程式的軟體。  
我記得在WM2003時，裝過Spb Pocket Plus，  
裡頭有個小執行檔還滿好用的。  
如果指定一個硬體按鍵給它，  
執行後就會跳出和Windows作業系統中大家熟悉的Alt-Tab畫面，  
趁畫面還沒結束時，再按一下的話，它就會依序去做切換。  
  
原本打算把這支小程式拿到Diamond或Touch HD上用的。  
結果…唯一能指定的功能鍵就是通話鍵長按。  
平常那麼常做的事（切換程式），要我每次都長按通話鍵，我大概怎樣也習慣不了吧。  
  
所以又開始上網亂找軟體。  
終於…讓我找到了一個好物！AE Button Plus  
有了它，HTC的手機就無敵了。  
不用再苦苦等他們員工哪天想通，做一個同樣好用的軟體，或是直接把key換掉。  
AE Button Plus讓你可以指定每個鍵的行為，包含單擊、雙擊、長按，或三擊。  
Back key可以設定，Talk/End key也可以設定。  
連上下左右鍵，機子左邊的音量鍵也行！（也是啦，平常誰沒事會去調音量）  
  
不過，Back key還是有點缺陷，在Today Screen時，似乎keyboard driver不會送出key event。  
所以如果手機回到Today Screen，Back key的設定會沒有用。（這是什麼behavior XD)  
這沒關係。因為靠它，就已經決解了我大半的問題。  
目前我的設定如下：  
VolumeUp  
Single Click: PageUp  
  
VolumeDown  
Single Click: PageDown  
Doubld Click: Volume Control  
  
Back Key  
Single Click: Task Manager (AE Button Plus內建的)  
Long Press: 時間 （也是它內建的，這樣子就不用想辦法讓時間一直出現在Title bar）  
  
Send Key  
Double Click: Start Menu  
這個我也一直覺得很不人性化。開始列做在畫面左上角是怎樣，以為每個人的手都很長嗎？  
明明按鍵設計都都是在手機下方，為什麼Start button和結束的x都要放在畫面的上方。  
做一件事，非得要整隻手機上下左右都摸一圈才行嗎？又不是打麻將。  
所以既然AE Button Plus可以指定執行Start Menu，當然要找個鍵來做，  
這樣子我才可以一手搞定所有事。 : )  
該touch的時候touch，不該touch的時候，就別想叫我用兩隻手。
