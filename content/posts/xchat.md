Every time I use xchat, I have to connect the irc server manually.  
And then, I have to choose the channels I want to join in all over again.  
That's really tiresome for me.  
  
so I did some search today abou this little trouble.  
Actually it's quite easy to solve if I pay attention to some docs on the net.  
  
1. choose the server list in the menu.  
2. edit the property of the very server you wanna connect:  
2.1 you'll find an option to make it automatically connect when the program is on.  
2.2 In the same dialog, you can find a place to input channels too.  
3. re-launch the program. voila, c'est tout!
