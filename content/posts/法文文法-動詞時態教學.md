法文主要有下列動詞時態：  
  
**l'indicatif 直陳式**  
  
le present 現在式  
  
[l'imparfait](http://netherlandsdaniel.blogspot.com/2007/10/french-limparfait.html) 未完成過去式  
[le passe simple](http://netherlandsdaniel.blogspot.com/2009/06/le-passe-simple.html) 簡單過去式   
  
le passe compose 複合過去式  
le passe anterieur  
le plus-que-parfait  
  
le futur simple  
le futur anterieur  
le futur dans le passe  
le futur anterieur dans le passe  
le passe immediat  
  
l'imperatif 命令式  
  
le conditionnel 條件式  
  
le subjonctif 虛擬式  
  
l'infinitif 不定式
