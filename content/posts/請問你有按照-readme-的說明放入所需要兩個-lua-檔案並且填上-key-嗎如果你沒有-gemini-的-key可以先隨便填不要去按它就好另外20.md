請問你有按照 readme 的說明，放入所需要兩個 lua 檔案，並且填上 key 嗎？如果你沒有 gemini 的 key，可以先隨便填，不要去按它就好。另外。2.0 的版本應該是要 run 在 development 版本上才不會在儲存筆記時出問題。你要進到 koreader 的menu > update 的設定，把 channel 改成 development，然後更新到 2024.04 之後更新的版本才可以。
